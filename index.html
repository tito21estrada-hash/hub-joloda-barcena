Operational Action Plans (Troubleshooting)

×
1. Safety Reset (E-Stop)

Condition: Emergency Stop button engaged. System halted.

1 Verify and clear the unsafe condition.
2 Twist red E-Stop button to release.
3 Press Blue Button (RESET) to rearm panel.
2. Physical Interlock Error (Gap)

Condition: Truck not properly docked against stopper sensors. System prevents transfer.

1 Identify failing sensor on HMI screen.
2 Request driver to re-align against stoppers.
3 Verify green interlock lights before proceeding.
3. Transfer Jam / Sensor Fault

Condition: Transfer stops mid-cycle due to pallet sensor error (crooked pallet/dirt).

1 Read specific Zone Error Code on HMI.
2 Inspect and clean floor sensor in affected zone.
3 Press Blue Reset and resume step-by-step.
